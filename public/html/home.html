JaggaCode
JaggaCode is Judge-a-code, an online platform which judges a solution before compiling it.

This platform is aimed to solve the high load problem on servers. The main idea is to implement certain checks that will be performed on the client side of the particular user before sending the submission to the server. Also these checks will not affect the performance of other users and the main server. Apart from this, we have some exciting and useful features such as code autosave, problem sorting, recommendation system and track user profile system.
